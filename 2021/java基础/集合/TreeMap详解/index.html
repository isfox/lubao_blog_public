TreeMap详解

TreeMap是Map接口的一个实现类，底层基于红黑树的实现

从继承机构可以看到TreeMap除了继承了AbstractMap类，还实现了NavigableMap接口，而NavigableMap接口是继承自SortedMap接口的，所以TreeMap是可以进行排序的
