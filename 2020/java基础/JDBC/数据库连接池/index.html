数据库连接池

数据库连接池的基本思想是为数据库建立一个”缓冲池”。预先在缓冲池中放入一定数量的连接，当需要建立数据库连接时，只需从”缓冲池”中取出一个，使用完毕再放入缓冲池。

JDBC的数据库连接池使用javax.sql.DataSource来表示，是一个接口，该接口通常由服务器提供实现，当然也有一些优秀的开源连接池

DataSource

DataSource通常被称为数据源，包含连接池和连接池管理两部分
