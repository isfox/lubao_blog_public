flume简单配置与执行

命令参数

  • agent 指定以agent角色启动，另一个角色为avro-client
  • conf或c 指定flume-env.sh和log4j.properties的所在目录
  • config-file或f 指定配置源和接收器配置文件的相对路径，相对于执行该命令的目录，包括文件名
  • name或n 指定agent的名称
  • D -D后接键值对，指定java相关配置
